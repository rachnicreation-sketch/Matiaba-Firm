# RM5 PLURI-SERVICES — Site Web

## Structure du projet

```
rm5-site/
├── index.html          ← Page principale (ouvrir dans un navigateur)
├── css/
│   └── style.css       ← Tous les styles CSS
├── js/
│   └── main.js         ← JavaScript : animations, formulaire, navigation
├── images/
│   ├── favicon.svg     ← Icône de l'onglet navigateur
│   ├── logo.svg        ← Logo (fond clair)
│   ├── logo-white.svg  ← Logo (fond sombre / footer)
│   ├── hero-pattern.svg ← Fond de la section héro
│   ├── about-visual.svg ← Visuel section "À propos"
│   ├── map.svg         ← Carte de localisation stylisée
│   └── icons.svg       ← Sprite d'icônes SVG
└── README.md           ← Ce fichier

```

## Comment utiliser

1. **Décompressez** l'archive ZIP dans un dossier de votre ordinateur.
2. **Double-cliquez** sur `index.html` pour ouvrir dans votre navigateur.
3. Le site fonctionne entièrement **sans serveur** (HTML/CSS/JS statique).

## Pour mettre en ligne

1. Souscrivez à un hébergeur web (ex: Hostinger, OVH, InfinityFree).
2. Uploadez TOUS les fichiers et dossiers via FTP ou le panneau de contrôle.
3. Votre site sera accessible via votre nom de domaine.

## Pour personnaliser

- **Textes** : Modifiez directement dans `index.html`
- **Couleurs** : Changez les variables CSS dans `css/style.css` (section `:root`)
  - `--red` : couleur principale (#E31E24)
  - `--dark` : couleur sombre (#0A0A0A)
- **Téléphone / Email** : Recherchez dans `index.html` et remplacez
- **Ajouter des photos** : Placez vos images dans le dossier `images/` et référencez-les dans le HTML
